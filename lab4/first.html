Каскадные таблицы стилей CSS (Cascading style sheets) — формальный язык описания внешнего вида документа,
созданного с использованием языка разметки гипертекста.

Каскадные таблицы стилей позволяют разделить описание логической структуры html-документа (выполненное с помощью
языка разметки) и описание внешнего вида html-документа (выполненное с помощью CSS)
